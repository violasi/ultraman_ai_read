# 汪汪队日记 使用说明

## 打开应用

双击 `index.html` 即可在浏览器中打开（推荐 Chrome）。

设备首次打开需要输入验证密码。

## 导入绘本

1.  打开应用后，点击底部导航栏的 **绘本馆**
2.  点击右上角 **管理**（或进入绘本管理页面）
3.  点击 **导入** 按钮
4.  选择 `.json` 绘本文件（支持多选），例如：
    -   `绘本系列-小羊上山-20本.json`
    -   `绘本系列-摩比汉语分级-45本.json`
5.  等待导入完成，绘本会自动出现在绘本馆中

## 设置已会汉字

进入 **设置**（右上角齿轮图标）已会汉字管理中可以编辑

也可以保留初始设置，在孩子阅读中，点击了任意一个初始已会汉字听了读音，该汉字就会来到生字表

生字表中有复习功能，请自由探索

## 设置奖励图片

角色陪读累计 3 天会解锁第 1 张奖励卡片，6 天解锁第 2 张，以此类推。

奖励图片存放在 `images/rewards/` 文件夹中，命名格式：

```         
{角色ID}_gift_{等级}.png
```

例如：`chase_gift_1.png`、`marshall_gift_2.png`

汪汪队版可用的角色 ID：
- `chase` - 阿奇
- `marshall` - 毛毛
- `skye` - 天天
- `rocky` - 灰灰
- `rubble` - 小砾
- `zuma` - 路马
- `everest` - 珠珠
- `tracker` - 小克
- `ryder` - 莱德

将准备好的 PNG 图片按命名规则放入 `images/rewards/` 即可。

## 数据备份

所有阅读记录、认字数据存在浏览器本地，清除浏览器数据会丢失。

进入 **设置**（右上角齿轮图标）可以导出/导入备份数据。
