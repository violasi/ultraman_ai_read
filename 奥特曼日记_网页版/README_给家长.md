# 奥特曼日记 · 儿童自主阅读小工具（使用说明）

## 这是什么？

这是一款为喜欢奥特曼的小朋友做的自主阅读小网页，用孩子感兴趣的奥特曼故事做内容，让阅读变得轻松又有成就感，慢慢养成自己看书的习惯。

## 三个阅读模式

- **汉字阅读**
  以常用汉字为主，不熟悉的字会自动标注拼音，点一下就能听发音，方便孩子自己读下去。
- **拼音阅读**
  全文带拼音，声母韵母用不同颜色标出，适合正在熟悉拼音的小朋友。
- **趣味阅读**
  简单易懂的短句故事，配合奥特曼情节，孩子更容易坚持读完。

## 智能阅读记录

工具会自动记录孩子的阅读情况：

- 孩子能顺畅读完的内容，会标记为熟悉内容
- 经常点读、不太熟的字词会自动收到"生词回顾"里
- 也可以手动调整已认识字词，更贴合孩子当前水平

## 奥特曼卡片收集（孩子最爱）

每读完一个小故事并完成小问答，就可以解锁一张奥特曼卡片，激励孩子继续阅读。

- 超多奥特曼卡片，包含经典角色与不同形态
- 连续阅读、完成故事可解锁成就与稀有卡片
- 集齐同一角色全套形态会有专属集齐标记

## 网页版使用方法

### 方式一：一键启动（最简单）

1. 解压下载好的文件包
2. 双击「启动奥特曼日记」文件
3. 浏览器会自动打开，直接使用

### 方式二：手动打开

1. 解压后找到 `dist/index.html`
2. 直接用浏览器打开即可使用
（部分语音朗读功能可能受浏览器限制）

## 日常怎么用

1. 每天打开一次，工具会自动推荐合适的故事
2. 让孩子自己选择汉字/拼音模式
3. 读完后有 3 道小问题，答对即可获得卡片
4. 不熟的字词可以点读发音，也可以加入回顾
5. 进度自动保存，下次打开继续阅读
6. 首页会显示连续阅读天数，鼓励坚持

## 故事工坊 · 生成新故事

故事看完了可以随时新增：

1. 打开底部「工坊」
2. 选择阅读模式、难度、喜欢的奥特曼角色
3. 点击一键复制提示词，粘贴到任意 AI 工具里生成故事
4. 复制 AI 生成的内容，回到工具里导入即可使用

也可以在设置中填入 API Key，实现一键生成故事。

## 存档与换设备使用

所有数据保存在你自己的设备上，换设备可以这样迁移：

1. 首页点击「存档」
2. 保存存档文件到本地
3. 新设备打开工具，读取存档文件即可恢复
4. 可预览新旧数据对比，确认后导入

## 小提示

- 所有内容仅保存在本地浏览器，不会上传云端
- 无广告、免费使用
- 适合喜欢奥特曼、想安静自主阅读的小朋友
